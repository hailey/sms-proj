Using our Public Matrix Server

Matrix is an open, decentralized communications protocol that allows you to communicate with any other srever that usees the matrix protocol.

Official Client @ app.thewords.faith.

You can connect to the matrix home server with any standard matrix client by setting your homeserver to thewords.faith. TheWords.faith matrix server should support email verification as well as VoIP and Video calls. Signups are open at the moment.

Any Matrix compattable client will work. If you woulda like to try another web client check out, Element Web, Schildi Chat Web, Cinny or my favorite fluffychat.im. For native apps I suggest SchildiChat for your phone or desktop. Or my favorite for smaller screens, fluffychat.im for android/ios.

For a list of all the clients available check out matrix clients.
